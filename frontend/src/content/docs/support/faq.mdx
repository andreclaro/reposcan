# Frequently Asked Questions

## General Questions

### What is SecurityKit?

SecurityKit is a comprehensive security audit platform that runs multiple industry-standard security scanners against your repositories and presents results in a unified dashboard.

### Is SecurityKit free?

Yes! We offer a free tier with:
- 5 scans per month
- Public repositories only
- Basic scanners (SAST, dependencies)
- Community support

[View all plans](/pricing)

### How is SecurityKit different from other security tools?

Unlike individual scanners, SecurityKit:
- Combines multiple scanners into one platform
- Provides unified results across all scanners
- Offers AI-powered analysis and prioritization
- Integrates seamlessly with GitHub and CI/CD

## Scanning Questions

### How long does a scan take?

| Repository Size | Typical Duration |
|-----------------|------------------|
| Small (< 10k lines) | 2-5 minutes |
| Medium (10k-100k) | 5-15 minutes |
| Large (100k+) | 15-30 minutes |

Factors affecting scan time:
- Number of scanners enabled
- Repository size
- Network speed (for cloning)
- AI analysis (adds 1-2 minutes)

### What scanners does SecurityKit use?

| Scanner | Purpose |
|---------|---------|
| Semgrep | SAST (static analysis) |
| Trivy | Container & filesystem |
| tfsec | Terraform security |
| Checkov | Infrastructure as Code |
| npm audit | Node.js dependencies |
| govulncheck | Go vulnerabilities |
| cargo-audit | Rust vulnerabilities |

### Can I run scans locally?

Yes! You can:
1. Use our [CLI tool](/docs/guides/cli)
2. Run the [Docker Compose stack](/docs/guides/docker)
3. Use the [API](/docs/api-reference)

### Do you store my code?

No. SecurityKit:
- Clones repositories temporarily for scanning
- Stores only scan results and metadata
- Deletes code after scan completes
- Never shares your code with third parties

## AI Analysis Questions

### What is AI analysis?

AI analysis uses artificial intelligence to:
- Generate risk scores (0-100)
- Provide executive summaries
- Prioritize findings
- Suggest fixes with effort estimates

### Is AI analysis available on all plans?

AI analysis is available on:
- ✅ Pro plan
- ✅ Enterprise plan
- ❌ Free plan (5 scans/month with AI)

### How does AI analysis work?

Our AI analyzes:
- All scan findings
- Code context
- Severity levels
- Historical patterns

Results in a risk score and prioritized recommendations.

### Is my code sent to AI providers?

No. Our AI processing:
- Uses anonymized findings data
- Never sends full source code
- Processes in secure environment
- Complies with SOC 2

## Troubleshooting Questions

### Why did my scan fail?

Common reasons:
- Repository is private (requires Pro)
- Invalid repository URL
- Repository doesn't exist
- Network issues
- Scanner errors

Check the scan logs in your dashboard for details.

### Why aren't findings showing up?

Possible reasons:
- No vulnerabilities detected (great news!)
- Scanner not enabled for repository
- File patterns excluded
- Scan didn't complete

### How do I suppress false positives?

Each scanner supports suppression:

**Semgrep:**
```javascript
// nosemgrep: rule-id
const query = `SELECT * FROM ${table}`;
```

**tfsec:**
```hcl
# tfsec:ignore:aws-s3-enable-bucket-encryption
resource "aws_s3_bucket" "public" {}
```

### Why is the AI analysis taking so long?

AI analysis typically takes 1-2 minutes. If it's longer:
- Large number of findings
- API rate limiting
- High server load

Try refreshing the page or regenerating the analysis.

# Welcome to SecurityKit

SecurityKit is a comprehensive security audit platform that helps you catch vulnerabilities before they reach production. Whether you're a solo developer or part of a large team, SecurityKit makes it easy to secure your code, dependencies, containers, and infrastructure.

## What is SecurityKit?

SecurityKit runs multiple industry-standard security scanners against your repositories and presents the results in a unified dashboard. Instead of learning and configuring multiple tools, you get everything in one place.

## Supported Scanners

| Scanner | Purpose | Languages/Formats |
|---------|---------|-------------------|
| Semgrep | Static analysis | All major languages |
| Trivy | Container & filesystem scanning | Docker, Kubernetes |
| tfsec | Terraform security | Terraform |
| Checkov | Infrastructure as Code | Terraform, CloudFormation, Kubernetes |
| tflint | Terraform linting | Terraform |
| npm audit | Node.js dependencies | Node.js, JavaScript |
| govulncheck | Go vulnerability database | Go |
| cargo-audit | Rust crate auditing | Rust |

## How It Works

1. **Connect** your GitHub repository
2. **Scan** runs automatically against your codebase
3. **Review** findings in the unified dashboard
4. **Fix** with AI-generated recommendations

# Features Overview

SecurityKit combines multiple industry-leading security scanners into a unified platform. Each scanner specializes in detecting different types of vulnerabilities.

## Scanner Categories

### 1. Static Application Security Testing (SAST)

Analyze your source code for security vulnerabilities without executing it.

- **SQL Injection**
- **Cross-Site Scripting (XSS)**
- **Hardcoded secrets**
- **Insecure cryptographic practices**
- **Logic flaws**

[Learn more about SAST →](/docs/features/sast)

### 2. Container Security

Scan Dockerfiles and container images for vulnerabilities and misconfigurations.

- **Base image vulnerabilities**
- **Insecure Dockerfile practices**
- **OS package vulnerabilities**
- **Sensitive data exposure**

[Learn more about Container Scanning →](/docs/features/containers)

### 3. Infrastructure as Code (IaC)

Validate your infrastructure definitions before deployment.

- **Terraform misconfigurations**
- **Cloud security best practices**
- **Compliance violations**
- **Resource exposure risks**

[Learn more about IaC Scanning →](/docs/features/infrastructure)

### 4. Dependency Auditing

Track vulnerable dependencies across multiple ecosystems.

| Ecosystem | Tool | What It Finds |
|-----------|------|---------------|
| Node.js | npm audit / pnpm audit | Known CVEs in npm packages |
| Go | govulncheck | Go vulnerability database entries |
| Rust | cargo-audit | RustSec advisories |
| Python | pip-audit | PyPI package vulnerabilities |

[Learn more about Dependency Auditing →](/docs/features/dependencies)

### 5. AI-Powered Analysis

Get intelligent insights beyond raw findings.

- **Risk scoring** (0-100)
- **Prioritized recommendations**
- **Executive summaries**
- **Fix effort estimation**

[Learn more about AI Analysis →](/docs/features/ai-analysis)

|---------|------|------------|-----|--------------|
